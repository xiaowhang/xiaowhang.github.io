训练模型的样本数据集检查

题目描述

在人工智能领域，通常需要对用于训练模型的样本数据集进行检查，如果有明显偏离样本的整体值的数据点（称为异常值，也称离群点，outlier)，就需要去掉或找一个新的数据替代这个离群点，以免影响训练模型的精度。通常可以用下述方法来检测样本的离群值。

对组成样本的数据 X={x0,x1,…,xN−1}X = \{x_0, x_1, \dots , x_{N-1}\}X={x0​,x1​,…,xN−1​} （总计 NNN 个数， 3≤N3 \le N3≤N ）进行抽样（抽取x中一部分数据组成新的数据集，例如把怀疑是离群点的数据去掉)。对新抽样样本 Y={y0,y1,…,yM−1}Y = \{ y_0, y_1, \dots ,y_{M-1} \}Y={y0​,y1​,…,yM−1​}（总计 MMM 个数， 2≤M≤N－12 \le M \le N－12≤M≤N－1），计算其平均值，

μ=1M∑i=0M−1yi\mu = \frac{1}{M}\sum^{M - 1}_{i = 0}y_iμ=M1​i=0∑M−1​yi​

和 YYY 的标准差，

σ=∑i=0M−1(yi−μ)2M−1\sigma = \sqrt{\frac{\sum^{M - 1}_{i = 0}(y_i - \mu )^2}{M - 1}}σ=M−1∑i=0M−1​(yi​−μ)2​​

对于原样本 XXX 中的数据 xix_ixi​ （其中 iii 为其在 XXX 的位置），如果满足 xi<μ−3σx_i \lt \mu - 3\sigmaxi​<μ−3σ 或者 xi>μ+3σx_i \gt \mu + 3\sigmaxi​>μ+3σ ，则可以认为 xix_ixi​ 是原样本数据集 XXX 的离群点。对于离群点需要找一个数替代它，使得样本 XXX 的标准差最小。

从键盘上输入 n (3≤n≤1000)n\ (3 \le n \le 1000)n (3≤n≤1000) 个实数组成一个数据集，使用上述方法确认其中是否含有离群点，如果没有输出 None.；反之则输出找到的第一个离群值（保留两位有效数字）、它在这个数据集中的位置和替代它的实数（保留两位有效数字）。
